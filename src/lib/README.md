# Library (`src/lib`)

This directory contains core business logic, utilities, and API definitions that are used across the application.

## Structure

- **osm/**: Contains all logic related to the Online Scout Manager (OSM) API, including the client, data parsers, and service functions.
- **auth.ts**: Authentication helpers and cookie management for OAuth tokens.
- **utils.ts**: General utility functions (e.g., Tailwind class merging).
- **api-definitions.ts**: Shared type definitions for API responses.
